- 游戏流程：
	- 起始发牌
		- 按顺序执行游戏：
			- 询问玩家：
				1. 是否叫卡波
					- 否：跳过并记录当前回合
				2. 从牌堆摸牌
					- 显示给当前玩家并询问：
						1. 是否弃牌
							- 结束本回合
						2. 如果有技能询问是否使用技能
							- 结束本回合
						3. 是否替换自己的手牌
							- 选择手牌的序号（至少一张）
							- 结束本回合
				3. 从弃牌堆摸牌（不为零）

					- 显示给当前玩家并询问：
						1. 是否弃牌
							- 结束本回合
						2. 是否替换自己的手牌
							- 选择手牌的序号（至少一张）
							- 结束本回合
		- 下一个玩家
