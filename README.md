- 牌堆
	- 生成卡牌：内容：数值，技能 

	- 取出牌顶
	- 检测是否为空
	- 清空牌堆


- 弃牌堆:
	- 显示牌顶
	- 放进卡牌
	- 去除卡牌

- 玩家:
	- 个人信息：
		- 名称， 编号，分数，手牌数

	- 显示当前手牌的状态
	- 叫卡波
	- 取出一张牌
	- 添加一张牌
	- 在牌尾加入一张牌
	- 反转牌面
	- 查看牌面
 
- 技能:
	- 四个
	- ......

- 游戏流程：
	- 起始发牌
		- 按顺序执行游戏：
			- 询问玩家：
				1. 是否叫卡波
					- 否：跳过并记录当前回合
				2. 从牌堆摸牌
				3. 从弃牌堆摸牌（不为零）

					- 显示给当前玩家并询问：
						1. 是否弃牌
							- 结束本回合
						2. 如果有技能询问是否使用技能
							- 结束本回合
						3. 是否替换自己的手牌
							- 选择手牌的序号（至少一张）
							- 结束本回合
		- 下一个玩家
